# Melhorias aplicadas ao site

Essas melhorias foram feitas para modernizar o site, melhorar a usabilidade, acessibilidade e o visual geral.

Resumo das alterações
- Atualização visual com tipografia moderna (Google Fonts Inter)
- Tema claro/escuro via `data-theme` com persistência em localStorage
- Header moderno, navegação, CTA e seção hero
- Cards para categorias (exercícios, guias, apostilas)
- Melhor listagem de arquivos com metadados (tamanho) e botões de download
- Pesquisa ao digitar (filtra arquivos em tempo real)
- Melhorias de acessibilidade: skip link, labels, foco visível e descrições ARIA
- Layout responsivo e melhorias no CSS (variables, dark theme)

Como testar localmente
1. Abra `index.html` no seu navegador (basta clicar duas vezes no arquivo ou arrastá-lo para o navegador).
2. Vá em `Exercícios` e teste a busca digitando termos como "lista", "funções" ou "matemática".
3. Clique no botão "Tema" para alternar entre claro e escuro (salva a preferência).
